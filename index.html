    The notification message you see above is produced by the jquery-attn.js plugin. 

         
    //Initialize a notification area for the 'body'. Messages will appear within <ul id="attn-container">...</ul>
    $('body').attn({
        container: '#attn-container'
    });

    //Add a success message to the notification area attached to 'body'.
    $('body').attn('success', '<strong>Welcome, stranger!</strong>');

    Of course you are not limited to delivering just good news to your users (the world ain't perfect). You can show error messages, warning messages, and info messages, too. 

        
    $('body').attn('error', '...'); //Displays an error message.
    
    $('body').attn('warning', '...'); //Displays a warning message.

    $('body').attn('info', '...'); //Displays an info message.

      You can also have messages that will fade-out after a specified period. Try it!

      The How: 

      
   $('body').attn('error', 'Your message here.', 3000); //Provide duration in milliseconds as the third parameter.

          Options

          Container

          These options must be set during initialization.

          Option Default Description
          container "#attn-container" Selector of <ul>...</ul> within the main selector. Messages will be appended to the target list.
          closeBtn "<a class="close">&times;</a>" HTML mark-up of close button.
          --- --- More to come.

          Messages

          These options can be defined to .attn() right after the message type. (Message types supported out-of-the-box are success, error, info, and warning. Adding of custom message types will be supported soon).

          Option Default Description
          message "" The message content.
          fade null If specified, the duration until the message fades out. This unit is in milliseconds (i.e, 1 second == 1000; 42 seconds == 42000).
          onClose $.noop (Callback function) Executed when the feedback-message is dismissed.
